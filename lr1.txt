Лабораторная работа №1
Пользователи и группы в linux

Цель: ознакомиться с правилами создания пользователей и групп. Изучить права доступа к обьектам в OC Linux

1.  Информация о пользователях доступна в файле /etc/passwd
Просмотр файла:
vim /etc/passwd
Укзаны мя пользователя, номер пользователя, номер группы, домашний каталог, командная оболочка  

Информация о группах доступна в файле /etc/group
vim /etc/group
Указаны имя групп и номер группы

Закрытая для пользователей информация о паролях доступна для аддминистратора в файле /etc/shadow
sudo /etc/shadow
Имя пользователя, пароль (или "!" в случае блокирования учетной записи), временные настройки учетной записи (время до смены пароля, время жизни записи и т.д.)

Закрытая для пользователей информация о парорлях для групп доступна для администратора в файле /etc/gshadow
"!" - означает отсуцтвие парорля для групп

Правила доступа к любому обьекту устонавливаються администратором при помощи комманды chmod
1 вариант:
sudo chmod 775 /home/tmp3 
Установили полный доступ к коталогу /home/tmp3 для владельца коталога и группы владельцев, а для всех остальных пользователей - право на просмотр (х) и чтение коталога (r)
