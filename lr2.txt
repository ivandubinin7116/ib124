Списки управления доступом (ACL)

ACL прав доступа - определяет доступ к файлу или коталогу на основе установленных прав
ACL по умолчанию - назначается только папке. Все обьекты внутри такой папки наследуют ACL родительской папки

Модификация прав доступна  осуществляется командой setfacl -m
Просмотр прав доступа - командой getfacl "имя обьекта"

Пример установки ACL (установка для чтения и выполнения для пользвателя tmp3):
setfacl -m u:tmp3:rx StickyBit/
student@alexander-VirtualBox:~$ ls -l

При смене разрешений командой chmod ACL сохроняются в прежнем виде, но сбрасывается mask
student@alexander-VirtualBox:~$ sudo chmod 755 StickyBit/
student@alexander-VirtualBox:~$ getfacl StickyBit/
# file: StickyBit/
# owner: student
# group: student
user::rwx
user:tmp3:r-x
user:tmp4:r-x
group::rwx			#effective:r-x
group:tempgrp:rwx		#effective:r-x
mask::r-x
other::r-x
 
Mask устанавливает эффективные максимальные права  доступа для всех пользователей и групп, пропиманных в ACL 
Пользователи сохпаняют свои настройки ACL, но работают только с правами, определенными настройками mask

Изминения  mask:student@alexander-VirtualBox:~/StickyBit$ setfacl -m u:tmp4:rwx ../StickyBit/

Исключение пользователя или группы из ACL:
student@alexander-VirtualBox:~/StickyBit$ setfacl -x u:tmp3 ./ (для пользователя)
Сброс всех ACL
student@alexander-VirtualBox:~/StickyBit$ setfacl -b tmp3 ./

Установка наследования по умолчанию:
student@alexander-VirtualBox:~/StickyBit$ setfacl -m d:u:tmp3:rwx ./

student@alexander-VirtualBox:~/StickyBit$ getfacl ./
# file: .
# owner: student
# group: student
user::rwx
group::rwx
other::r-x
default:user::rwx
default:user:tmp3:rwx
default:group::rwx
default:mask::rwx
default:other::r-x
